# Chat App

Real-Time Chat App with Socket.io

### Quick start

```bash
# clone the repo
git clone https://github.com/semihsemih/Chat-App.git

# change into the repo directory
cd Chat-App

# install 
npm install

# run
npm start
```

Then visit [http://localhost:3000](http://localhost:3000) in your browser.
